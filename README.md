# World Discover Poster (VRChat World紹介ポスター自動生成)

X（旧Twitter）に投稿されたVRChatのワールド紹介ツイートを自動でキャプチャし、一枚のポスター画像（一覧画像）として合成・出力するシステムです。
GitHub Actions と GitHub Pages を利用して、定期的な自動生成とデプロイを完全自動で行います。

## ✨ 主な機能

- **マルチプラットフォーム対応**: `PC` 向けワールドと `Quest` 向けワールドを別々に収集・合成します。
- **ダークモード / ライトモード対応**: キャプチャ時にテーマを自動で切り替え、2種類のデザインのポスターを生成します。
- **完全自動化**: GitHub Actions により、定期的に（あるいはトリガー時に）最新のツイートを自動で取得・画像化します。

## 🛠 使用技術

- **Node.js**: アプリケーション本体
- **Puppeteer**: X (Twitter) の埋め込みツイートをヘッドレスブラウザでレンダリング＆スクリーンショット
- **@napi-rs/canvas**: 取得した複数枚のスクリーンショットを1枚のポスター画像に高速で合成
- **GitHub Actions**: スクリプトの自動実行と画像ファイルの継続的蓄積
- **GitHub Pages**: 生成されたポスター画像と JSON データを静的ホスティング

## 🚀 動作の仕組み

1. `work/posterData.json`（外部サーバーから取得した対象ツイート一覧）を読み込みます。
2. Puppeteerが各ツイートのURLをヘッドレスブラウザで開き、スクリーンショットを撮影します。
3. 前回実行時に取得した過去の画像をロードしつつ、新しい画像と組み合わせてタイル状（2×3など）に合成・上書きします（`@napi-rs/canvas` を使用）。
4. 完成した画像群（`images` フォルダ）とインデックス情報（`posterInfo.json`）を GitHub Pages にデプロイします。

## ⚙️ セットアップと設定

このリポジトリは GitHub Actions 上で動作するように設計されているため、特別なローカル環境の構築なしで運用が可能です。
